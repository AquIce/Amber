# Aqu1

## Initialization

Ensure you have internet connection, and run the following script from the root of the repo:
```sh
./scripts/init.bat
```
This will pull the SDL.dll file from the SDL github repository.

## Build

To build the project, use the `build.bat` script:
```sh
./scripts/build.bat
```
